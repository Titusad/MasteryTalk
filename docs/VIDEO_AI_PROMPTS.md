# 🎬 Prompt para Generación de Video con IA

> Usa este prompt con herramientas como **Vidu, Synthesia, HeyGen, o Runway** para generar el video tutorial de inFluentia PRO.

---

## Prompt Principal (Copiar y Pegar)

```
Create a professional product tutorial video for "inFluentia PRO", an AI-powered English coaching platform for Latin American professionals.

STYLE:
- Clean, modern SaaS product demo style (similar to Notion, Linear, or Vercel product videos)
- Dark UI theme with teal/cyan accent colors and subtle glassmorphism effects
- Smooth, cinematic screen transitions (fade, zoom-in on key elements)
- Minimalist text overlays in white sans-serif font (Inter or similar)
- Background music: subtle lo-fi or ambient corporate, low volume

STRUCTURE (4:30 total):

SCENE 1 — HOOK (0:00–0:15)
Visual: Split screen showing a nervous professional on a video call (left) vs. a confident professional speaking fluently (right). Transition to the inFluentia PRO logo with tagline "Prepare. Practice. Perform."
Text overlay: "Your next big conversation deserves professional preparation."

SCENE 2 — SIGN UP (0:15–0:30)
Visual: Screen recording showing the landing page. User clicks "Comenzar gratis", Google OAuth screen appears, user logs in. Dashboard appears with "¡Buenos días!" greeting.
Text overlay: "Sign up free with Google. No credit card needed."

SCENE 3 — CHOOSE SCENARIO (0:30–1:00)
Visual: Screen recording of clicking "Comenzar práctica". Scenario cards appear (Job Interview, Sales Pitch, Client Meeting). User selects "Job Interview". Form fields appear for company, role, job description.
Text overlay: "Tell the AI about YOUR real scenario."
Highlight: AI suggestion bubbles appearing to autocomplete fields.

SCENE 4 — PRE-BRIEFING (1:00–1:40)
Visual: Loading animation, then the pre-briefing screen appears with a generated script. Zoom into key sections: Value Pillars, Power Phrases, Objection Handling.
Text overlay: "AI generates a personalized strategy — your conversation roadmap."
Animation: Bullet points appearing one by one with a subtle glow effect.

SCENE 5 — LIVE PRACTICE (1:40–2:30)
Visual: Voice practice interface. AI message bubble appears with audio waveform playing. User's microphone activates, speech-to-text appears in real time. Back-and-forth conversation (3 turns shown).
Text overlay: "Practice the real conversation with your AI interviewer."
Key moment: Show the pronunciation score appearing subtly in the corner after user speaks.

SCENE 6 — FEEDBACK ANALYSIS (2:30–3:20)
Visual: Analyzing screen with spinning animation → Feedback screen appears. Three pillars (Fluency, Vocabulary, Professional Communication) with scores. Zoom into a "Before/After" comparison showing the user's actual phrase vs. the improved version.
Text overlay: "See exactly what you said vs. how to say it better."
Animation: Before (red/orange highlight) morphing into After (green highlight).

SCENE 7 — GOLDEN SCRIPT (3:20–3:50)
Visual: Recap screen showing the "Golden Master Script" — polished version of everything said. User clicks "Download PDF". PDF preview appears.
Text overlay: "Take your improved script to the real meeting."

SCENE 8 — DASHBOARD PROGRESS (3:50–4:15)
Visual: Dashboard with populated data. Skill radar chart animating. CEFR level badge. Session history cards. "Phrases to Review" section with spaced repetition cards.
Text overlay: "Track your progress. Review key phrases daily."
Animation: Radar chart arms extending outward.

SCENE 9 — CTA (4:15–4:30)
Visual: Zoom out to full landing page. "Comenzar gratis" button pulses with a glow effect. URL appears: influentiapro.com
Text overlay: "Your first session is free. Start practicing today."
Logo + tagline fade in.

COLOR PALETTE:
- Background: #0f172a (dark navy)
- Primary: #06b6d4 (teal/cyan)
- Accent: #22d3ee (light cyan)
- Success: #10b981 (green)
- Warning: #f59e0b (amber)
- Text: #f1f5f9 (light gray)
- Cards: rgba(30, 41, 59, 0.8) with blur backdrop

TYPOGRAPHY:
- Headings: Inter Bold
- Body: Inter Regular
- Code/metrics: JetBrains Mono

VOICE OVER (Spanish, Latin American accent):
- Tone: Professional but warm, like a knowledgeable colleague
- Pace: Moderate, clear enunciation
- Gender: Male or female, mature professional voice

MUSIC:
- Style: Ambient corporate / lo-fi beats
- Volume: 15-20% during voiceover, 40% during transitions
- Reference: "Productize" by Epidemic Sound or similar
```

## Prompt para Versión Corta (Reels/TikTok — 60 segundos)

```
Create a 60-second vertical video (9:16) for Instagram Reels promoting "inFluentia PRO".

STYLE: Fast-paced, dynamic, trendy. Dark UI with cyan accents. Quick cuts. Kinetic text animations.

STRUCTURE:
0-5s: Hook — "Tu próxima entrevista en inglés? Prepárate con IA." (text slams onto screen)
5-15s: Quick montage of setup — choosing scenario, filling context
15-30s: Live practice — AI speaking, user responding, real-time transcription
30-45s: Feedback — Before/After comparison with dramatic reveal
45-55s: Golden Script + Dashboard progress tracking
55-60s: CTA — "Primera sesión GRATIS" + URL

MUSIC: Trending lo-fi or upbeat instrumental. Fast tempo.
TEXT: Large, bold, centered. Spanish. Motion graphics.
NO VOICEOVER — text-driven with sound effects.
```

## Prompt para Thumbnail de YouTube

```
Create a YouTube thumbnail (1280x720) for an inFluentia PRO tutorial video.

Elements:
- Left side: Screenshot of the app's voice practice screen (dark UI with message bubbles)
- Right side: Confident professional with headphones, smiling
- Bold text overlay: "PRACTICA INGLÉS CON IA"
- Subtitle: "Tu coach virtual de inglés profesional"
- inFluentia PRO logo in bottom-right corner
- Color scheme: Dark navy background, cyan/teal accents, white text
- Style: Clean, professional, high contrast

Do NOT include: play buttons, red arrows, shocked faces, clickbait elements
```

## 📌 Notas de Uso

| Herramienta | Mejor Para | Prompt a Usar |
|-------------|-----------|---------------|
| **Synthesia / HeyGen** | Video con avatar AI hablando | Prompt principal + voiceover |
| **Runway Gen-3** | Clips cinematográficos de transición | Escenas 1 y 9 |
| **Vidu** | Animaciones de UI/producto | Escenas 3-8 |
| **CapCut** | Edición final + subtítulos | Combinar todos los clips |
| **Midjourney / DALL-E** | Thumbnail de YouTube | Prompt de thumbnail |
| **ElevenLabs** | Voz en off en español | Texto del guión del video |

> **Recomendación:** Graba screen recordings reales de la app y usa IA solo para las transiciones, la voz en off, y los elementos gráficos adicionales. El resultado será más auténtico.
